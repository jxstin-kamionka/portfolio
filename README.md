# 💼 Portfolio – Justin Kamionka

Willkommen in meinem Portfolio! Dieses Projekt dient als persönliche Webseite, auf der ich mich, meine Fähigkeiten und meine Projekte präsentiere.

## 👋 Über mich

Ich bin **Justin Kamionka** und interessiere mich für Webentwicklung, IT-Lösungen und Prozessoptimierung. Neben meinem Beruf als Werkzeugmechaniker entwickle ich moderne Webanwendungen und arbeite kontinuierlich daran, mein Wissen in Frontend- und Backend-Technologien auszubauen.

## 🚀 Features

* Responsive Design für Desktop, Tablet und Smartphone
* Übersicht über meine Projekte
* Informationen zu meinem beruflichen Werdegang
* Darstellung meiner technischen Fähigkeiten
* Kontaktbereich
* Moderne und benutzerfreundliche Oberfläche

## 🛠️ Verwendete Technologien

* HTML5
* CSS3
* JavaScript (ES6+)
* Git & GitHub

## ⚙️ Installation

Repository klonen:

```bash
git clone https://github.com/DEIN-GITHUB-NAME/portfolio.git
```

In den Projektordner wechseln:

```bash
cd portfolio
```

Anschließend die `index.html` im Browser öffnen oder einen Live-Server verwenden.
